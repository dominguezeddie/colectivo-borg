## Architecture: The Three Roles of the Mother Cell

### Trust Rank (Gold/Silver/Bronze)

Measures node **consistency and reliability**. It does not depend on hardware. It is built through participation history.

| Rank | Condition (participation only) | Benefit |
|------|--------------------------------|---------|
| 🥇 **GOLD** | Score ≥ 80 | Maximum priority for microtasks. Chooses first. |
| 🥈 **SILVER** | Score 40-79 | Standard participation. |
| 🥉 **BRONZE** | Score < 40 (or new node) | Subject to audits. Receives tasks when available. |

**Basic Contribution Score formulas:**

| Event | Score Change |
|-------|---------------|
| New node (first boot) | Score = 100.0 |
| Microtask completed successfully | +0.5 |
| Microtask failed / timeout | -15.0 |
| Validator audit (honest node) | +5.0 (every 100 tasks) |
| Validator audit (fraudulent node) | -30.0 + possible isolation |

> *"Reputation is earned through consistency, not purchasing power."*

### Capacity Profile (High/Medium/Basic)

Measures raw **hardware power**. Automatically detected at each boot. Does not affect trust rank.

| Profile | Condition | Which tasks it receives |
|---------|-----------|-------------------------|
| **HIGH** | RAM ≥ 4GB or modern CPU (2018+) | Heavy tasks (local inference, aggregation) |
| **MEDIUM** | RAM ≥ 2GB, CPU 2014-2017 | Standard tasks (classification, short transcription) |
| **BASIC** | RAM < 2GB or CPU pre-2014 | Light tasks (sums, validations, handshakes) |

### Practical combination

- A **Gold + Basic** node (old PC always on) gets high priority for light tasks and handshakes. It is the network's backbone.
- A **Bronze + High** node (new PC but rarely used) gets heavy tasks when available, but without priority.
- The network does not penalize limited hardware. It only measures real commitment.

## Hardware (Updated for 2026)

The Borg targets equipment manufactured between **2016 and 2020** — machines still circulating in schools, libraries, homes, and cooperatives.

| Device | Minimum requirement | RAM | Role |
|--------|---------------------|-----|------|
| Desktop / Laptop PC | 2015 onward, 64-bit CPU | 4 GB | Executor, light Aggregator |
| Android phone | 2018 onward (Android 8+) | 3 GB | Mobile Executor, itinerant node |
| Raspberry Pi / SBC | Raspberry Pi 3B+ onward | 1-4 GB | Light Lighthouse Node |
| Very old PC (optional) | 2010-2014, 2 GB RAM | 2 GB | Executor for minimal microtasks |

> *"A cell phone can contribute minutes. A laptop can contribute hours. An old PC can become a permanent node. Everyone participates according to their possibilities."*

### Preloaded Knowledge Base

The Embryo USB drive includes a portable library (<500 MB) with local dictionaries, agriculture manuals, first aid guides, WISP documentation, and public domain texts. The first node has immediate value even without a network.
