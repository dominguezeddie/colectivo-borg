## ¿Qué es el Colectivo Borg?

El **Colectivo Borg** es una red de inteligencia artificial distribuida, open source y ecológicamente honesta, diseñada para el **60% de la humanidad** que no tiene acceso real a las IAs corporativas.

No es una empresa. No tiene accionistas. No tiene servidor central. No tiene términos de servicio que puedan cambiar mañana.

Es infraestructura cognitiva colectiva — como Wikipedia, pero para el cómputo distribuido de IA.

---

## El problema que resuelve

| Problema | Impacto real |
|---|---|
| **Consumo energético** | Un data center de IA consume tanta electricidad como una ciudad mediana |
| **Monopolio corporativo** | 3-4 empresas deciden quién accede y en qué condiciones |
| **Exclusión del 60%** | Costo en monedas fuertes, conectividad insuficiente, barreras lingüísticas |
| **Peaje lingüístico** | El español consume un **59% más de tokens** que el inglés — los hispanohablantes pagan más por el mismo resultado |

Cuando las corporaciones racionan capacidad, el 60% lo nota primero. El Borg resuelve esto por diseño: **sin tokens, sin escasez, sin dueño.**

| Modalidad | Descripción | Dueño del equipo |
|---|---|---|
| **Borg Portable** | Pendrive booteable. RootFS on RAM. No toca el disco. Identidad portátil. | El usuario |

## Instalación y Primer Nodo

```bash
# 1. Clonar el repositorio
git clone https://github.com/colectivo-borg/colectivo-borg.git
cd colectivo-borg

# 2. Iniciar el nodo Ejecutor (Terminal 1)
python main.py

# 3. Enviar una microtarea de prueba (Terminal 2)
python emisor.py --texto "La casa era muy grande y tenía jardín."
```

**Variables de entorno opcionales:**

```bash
BORG_NOMBRE=mi-nodo-01   # Nombre del nodo (default: hostname)
BORG_HOST=0.0.0.0        # Interfaz a escuchar
BORG_PUERTO=65432        # Puerto TCP
BORG_CICLO=4.0           # Segundos de pausa entre ciclos (control térmico)
```

> **Control térmico:** El ciclo de 4 segundos es crucial para CPUs de 2011.
> Evita busy-waiting y mantiene el hardware antiguo frío y estable.
> Ajustar a 0.5 en hardware moderno si se desea mayor throughput.

## Filosofía: Por Qué el Borg es Incorruptible

Las IAs corporativas pueden ser manipuladas para el beneficio de sus dueños sin cuestiones éticas ni morales. El Borg es estructuralmente inmune:

| Aspecto | IA Corporativa | Colectivo Borg |
|---|---|---|
| Dueño | Una corporación con accionistas | Nadie. La red es de todos. |
| Código | Cerrado y secreto | Abierto — auditable por cualquiera |
| Incentivo | Maximizar beneficio corporativo | Bien común |
| Sesgo | Optimizado para el dueño | Auditado por miles sin interés corporativo |
| Corrupción | Posible en silencio | Imposible — el código es público |
| Adquisición corporativa | Posible | **Imposible** — sin activo central que comprar |

> *"El Borg no puede ser comprado. No hay activo central que adquirir."*

La transparencia radical es la garantía ética. No una promesa — una consecuencia estructural de la arquitectura abierta.
